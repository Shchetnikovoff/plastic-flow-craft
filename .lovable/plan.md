### Описание товара
- Название: "Отвод 90° вентиляционный круглый: раструб"
- Описание назначения
- Используемые материалы (PPC, PE 100, PPH, PPs)
- Характеристики: диаметр 200–1200 мм, соединение — раструб

### Схемы
- Простая SVG-схема отвода 90° с обозначением размеров (D, раструб)

### Таблица типоразмерного ряда
- Столбцы: Артикул, Типоразмер трубы, Толщина стенки, Возможная длина, Толщина раструба, Количество, Действие
- 18 строк по данным из таблицы пользователя (200–1200 мм)
- Артикулы генерируются автоматически (формат: ОТВ-90-РС-XXX, где XXX — диаметр)
- Каждая строка кликабельна (подсветка при наведении)
- В каждой строке: поле ввода количества (+/−) и кнопка "В корзину"
- Цена — "По запросу"

### Корзина
- Иконка корзины в шапке с счётчиком товаров
- При клике — выдвижная панель со списком добавленных позиций
- Кнопка "Отправить заявку" — формирует заявку и показывает контактную информацию

### Страницы и компоненты
- **Header** — логотип, название компании, контакты, корзина
- **ProductPage** — галерея, описание, схема, таблица
- **Cart** — выдвижная панель (Sheet) с позициями
- Всё на одной странице, без бэкенда, состояние корзины в React state
