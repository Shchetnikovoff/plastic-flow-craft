

# План: Скрубберы вертикальные — удаление горизонтальных + кликабельные строки с КП и спецификацией

## Что будет сделано

1. **Создание файла данных `src/data/scrubberProducts.ts`**
   - Интерфейс `ScrubberProduct` (article, name, model, flow, dimensions, image)
   - Массив из 17 вертикальных моделей серии СН с артикулами формата `СЗПК.СН.{flow}` (СН-0.1 → СН-40)

2. **Переработка страницы `src/pages/GazoochistkaSkrubbery.tsx`**
   - Удаление всего, связанного с горизонтальными скрубберами: массив `ctModels`, вкладка "Горизонтальные (серия СТ)", изображение горизонтального скруббера из Hero-секции, упоминания "горизонтальные" из текстов
   - Удаление компонента `Tabs` — остаётся единственная таблица вертикальных моделей
   - Добавление столбца "Артикул" в таблицу
   - Строки таблицы становятся кликабельными (`cursor-pointer`, `hover:bg-accent/50`) с навигацией на `/product/{article}`
   - Удаление `useSearchParams` (вкладки больше нет)

3. **Добавление карточки товара в `src/pages/Product.tsx`**
   - Поиск по артикулу в `scrubberProducts`
   - Карточка со спецификацией: модель, производительность, габариты, материал
   - Кнопка "Скачать спецификацию (PDF)" через `generateSpecPdf`
   - Кнопка "Добавить в КП" через `addToKp`
   - Хлебные крошки: Каталог → Газоочистка → Скрубберы → {артикул}

4. **Обновление каталога `src/data/catalog.ts`**
   - Удаление подкатегории "Скрубберы горизонтальные"
   - Обновление пути подкатегории "Скрубберы вертикальные" на `/catalog/gazoochistka/skrubbery` (без `?tab=vertical`)
