B: Табличный формат

Все товары в таблице: миниатюра + артикул + спеки в строку. Компактно, классический формат КП. Доп. строки вручную.

Гибрид: Карточка для одного товара, Таблица для нескольких

КП на один товар — формат A (крупная карточка с фото). КП из корзины (несколько товаров) — формат B (таблица). Лучшее из обоих миров.
